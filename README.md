# Rumah Sakit Terlarang

Game 3D Horror Survival Puzzle berbasis browser menggunakan Three.js.

## Tech Stack
- HTML5, CSS3, JavaScript (ES6+ Modules)
- Three.js

## Cara Menjalankan
1. `npm install`
2. `npm run dev` (Menggunakan Vite) atau `npm start` (Http-server)
3. Buka browser di `http://localhost:8080` atau port yang ditentukan.
